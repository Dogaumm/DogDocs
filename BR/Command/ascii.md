# Comando de Cifras

## Nome do comando
> **ascii**

## Descrição
> Este comando consegue transformar e traduzir cifras em Ascii.

## Argumentos
- texto - `Obrigatório` - É obrigatório caso queira transformar em Ascii.

## Exemplo
> `$ascii -t oi, tudo bem?`

> `$ascii 111 105 44 32 116 117 100 111 32 98 101 109 63`
