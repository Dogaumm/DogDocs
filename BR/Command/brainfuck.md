# Comando de Cifras

## Nome do comando
> **brainfuck**

## Descrição
> Este comando consegue transformar e traduzir cifras/a linguagem de programação em Brainfuck.

## Argumentos
- texto - `Obrigatório` - É obrigatório caso queira transformar em Brainfuck.

## Exemplo
> `$brainfuck -t oi, tudo bem?`
