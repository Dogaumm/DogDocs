# Comando Secundário

## Nome do comando
> **traduzir**

## Descrição
> Este comando serve para mostrar algumas informações e estatísticas do bot.

## Argumentos
- texto - `Obrigatório` - Um texto para o bot traduzir

## Exemplo
> `$traduzir text`

> * `Hi guys!`
> * `$traduzir`
> * `Bot: Olá galera!`
